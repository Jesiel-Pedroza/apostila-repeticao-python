Estruturas de Repetição em Python

👋 Bem-vindo à sua jornada de aprendizado em programação com Python!

Nesta apostila interativa, você vai entender um dos pilares da lógica de programação: as estruturas de repetição.

Com elas, conseguimos automatizar tarefas, criar laços de execução e construir sistemas muito mais eficientes.

Vamos começar? 🚀✨

🔄 O que são?

As estruturas de repetição permitem executar um bloco de código várias vezes, de forma controlada.

  • 🔁 while: repete enquanto uma condição for verdadeira.
  • 📚 for: percorre uma sequência (como listas ou intervalos).

Essas estruturas são fundamentais para automatizar processos, como percorrer listas, validar entradas ou executar cálculos.

💡 Por que usar?

  • 🔁 Automatiza tarefas repetitivas e manuais.
  • 🧹 Torna o código mais limpo e legível.
  • ⚡ Economiza tempo e melhora a performance.
  • 🧠 Permite criar soluções inteligentes e adaptáveis.

Imagine ter que digitar a mesma instrução 100 vezes. Com uma estrutura de repetição, isso vira apenas 3 linhas de código!

👨‍💻 Exemplos Explicados

🔂 Exemplo com while:

Repete enquanto a condição for verdadeira:

contador = 1
while contador <= 5:
  print("Contando:", contador)
  contador += 1

🔁 Exemplo com for:

Percorre um intervalo definido:

for i in range(1, 6):
  print(i)

➕ Exemplo com soma acumulada:

soma = 0
for i in range(1, 11):
  soma += i
print("Soma:", soma)

🎯 Validação com while:

nota = -1
while nota < 0 or nota > 10:
  nota = float(input("Digite uma nota entre 0 e 10: "))
print("Nota válida!", nota)

📊 Quando usar while ou for?

Situaçãowhilefor
Condição incerta (ex: até digitar "sair")✔️
Laço com fim conhecido (ex: 10 vezes)✔️
Validação de dados✔️
Percorrer lista, string ou intervalo✔️

Dica: Sempre que souber quantas vezes o laço deve ocorrer, prefira o for. Caso contrário, use while.
